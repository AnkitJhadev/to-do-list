## Requirements

- Add new tasks when clicking the **"Submit"** button.
- Remove tasks when clicking the **"Delete"** button.
- Use **React** (functional components & hooks).
- Ensure state management is handled efficiently.
- Implement event handling properly.
- Follow clean code and component reusability practices.
- _(Optional)_ Add basic CSS styling if time permits.

---

## Evaluation Criteria

- The app should be functional and allow adding/removing tasks dynamically.
- Code should be readable, maintainable, and follow React best practices.
- Use of controlled components for input fields.
- Minimal external dependencies (except for React itself).
